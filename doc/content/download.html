Which deployment is right for me?

Cloud

If you're looking for ease of use in an enterprise grade LoRaWAN solution, {{% tts %}} Cloud is a great option. You get a managed tenant on a shared cluster, backed by our Guaranteed uptime agreement for worry-free deployment with automatic updates forever. Your deployment scales automatically to support as many devices as you need, all managed seamlessly in the cloud from anywhere in the world.

Dedicated Cloud

For those who need dedicated CPU power or need a dedicated machine for security reasons. All the benefits of Cloud, on a dedicated cluster.

AWS Launcher

If you're familiar with the AWS ecosystem, our AWS Launcher makes it easy to deploy in one click on your new or existing EC2 cluster. Integrates effortlessly with AWS IOT, so you can keep using tools you're already familiar with.

Enterprise

Meet data privacy concerns by bringing your own hardware. Professional support to deploy, backed by our Guaranteed uptime agreement and 24/7 phone service.

The Things Network

Easily test devices and use cases on our free community network.

Open Source

For DIY'ers, the core of {{% tts %}} is open source. If you want to get your hands dirty and understand how {{% tts %}} works, clone away and get hacking.
